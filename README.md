# ReallyEC

Плагин для Spigot/Paper, который заменяет ванильный эндер-сундук на расширяемое хранилище до 54 слотов.

## Что умеет

- Команды `/ec`, `/enderchest`, `/echest`, `/ender`, `/enderc`.
- При команде открывается меню выбора: открыть эндер-сундук или открыть улучшения.
- При клике по обычному блоку эндер-сундука открывается тот же кастомный эндер-сундук игрока.
- Изначально доступно 27 слотов.
- Улучшения до 36, 45 и 54 слотов.
- Цена улучшений задаётся предметами в `config.yml`.
- Все названия, материалы, лор, слоты меню, заголовки и сообщения редактируются в `config.yml`.
- Хранилища игроков сохраняются в `plugins/ReallyEC/players/`.

## Сборка

Нужны Java 8+ и Maven.

```bash
mvn package
```

Готовый `.jar` появится в папке `target/`.

## Сборка на GitHub

В проекте уже есть workflow `.github/workflows/build.yml`.
После загрузки проекта на GitHub сборка запускается при `push`, `pull request` и вручную через вкладку `Actions`.
Готовый `.jar` можно скачать в артефактах workflow под названием `ReallyEC-jar`.

## Права

- `reallyec.use` - пользоваться меню и эндер-сундуком, по умолчанию доступно всем.
- `reallyec.reload` - перезагрузка конфига через `/reallyec reload`, по умолчанию только OP.

## Настройка улучшений

Пример блока улучшения:

```yml
upgrades:
  "36":
    slot: 20
    material: ENDER_CHEST
    name: "&dУлучшение до 36 слотов"
    lore:
      - "&7Открывает ещё один ряд."
    requirements:
      ENDER_PEARL: 16
      OBSIDIAN: 8
```

Ключ `"36"` означает, сколько слотов станет доступно после покупки.
