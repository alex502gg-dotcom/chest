# ReallyEC

Плагин для Spigot/Paper: расширяемый эндер-сундук до 54 слотов, меню как на ReallyWorld, улучшения за деньги EssentialsX Economy, invsee и админ-очистка.

## Возможности

- `/ec`, `/enderchest`, `/echest`, `/ender`, `/enderc` открывают главное меню.
- Обычный блок эндер-сундука открывает этот же кастомный EC.
- Стартовый размер: 27 слотов.
- Улучшения: 36, 45, 54 слота.
- Оплата улучшений деньгами EssentialsX Economy.
- Меню улучшений показывает зелёные разблокированные слоты и красные заблокированные.
- Перед покупкой открывается меню подтверждения: по центру этап и цена, зелёное стекло - купить, красное - назад.
- `/ec invsee <игрок>` открывает EC другого игрока.
- Игроки с `ecrw.invsee.player` только смотрят чужой EC.
- Админы с `ecrw.invsee.admin` могут забирать и перекладывать вещи в чужом EC.
- `/ec clear upgrade <игрок|*>` сбрасывает улучшение до начального этапа и полностью чистит EC.
- `/ec clear inv <игрок|*>` чистит только содержимое EC.
- `/ec reload` перезагружает конфиг.
- Все названия, материалы, слоты меню, лор, цены и сообщения редактируются в `config.yml`.

## Требования

- Java 8+
- Spigot/Paper 1.16.5+
- EssentialsX с включённой экономикой

## Сборка

```bash
mvn package
```

Готовый `.jar` появится в папке `target/`.

## Сборка на GitHub

В проекте есть workflow `.github/workflows/build.yml`.
После загрузки проекта на GitHub сборка запускается при `push`, `pull request` и вручную через вкладку `Actions`.
Готовый `.jar` можно скачать в артефактах workflow под названием `ReallyEC-jar`.

## Права

- `ecrw.*` - все права плагина.
- `ecrw.use` - пользоваться `/ec` и эндер-сундуком.
- `ecrw.reload` - `/ec reload`.
- `ecrw.clear` - `/ec clear upgrade ...` и `/ec clear inv ...`.
- `ecrw.invsee.player` - смотреть чужой EC без возможности брать вещи.
- `ecrw.invsee.admin` - смотреть и редактировать чужой EC, можно забирать вещи.

## Настройка цены

```yml
upgrades:
  "36":
    slot: 22
    material: ENDER_CHEST
    name: "&dУлучшение до 36 слотов"
    lore:
      - "&7Открывает ещё один ряд."
    price: 25000
```

Ключ `"36"` означает, сколько слотов станет доступно после покупки.
